    <article class="markdown-body entry-content" itemprop="text"><h1><a id="user-content-lets-learn-our-state-capitals" class="anchor" aria-hidden="true" href="#lets-learn-our-state-capitals"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Let's Learn our State Capitals</h1>
<p>We're going to create a game to help us memorize the names of the capitals of all 50 states.</p>
<p>To play the game:</p>
<ul>
<li>Your program should prompt the user to identify the capital associated with a given state.</li>
<li>There should be running tallies on the number of correct and incorrect answers for each state</li>
<li>After getting through all 50 states one time, users should be asked whether or not they want to play again.</li>
</ul>
<h3><a id="user-content-game-requirements" class="anchor" aria-hidden="true" href="#game-requirements"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Game Requirements</h3>
<ul>
<li>
<p>Make sure the states don't appear in alphabetical order in the prompts. This will make the game a bit more challenging for the user.</p>
</li>
<li>
<p>Provide a welcome message to introduce the player to the game.</p>
</li>
<li>
<p>Initialize new keys in the dictionaries that store the number of times a user gets a capital <code>correct</code> and the number of times the answer is <code>wrong</code>.</p>
</li>
<li>
<p>Through all 50 states, prompt the user to name the capital of the state.</p>
</li>
<li>
<p>If the answer is correct, display a message saying so, and increment the <code>correct</code> key.</p>
</li>
<li>
<p>If the answer is wrong, display a message saying so, and increment the <code>wrong</code> key.</p>
</li>
<li>
<p>After each prompt, display a message telling the reader how many times the state was answered correctly out of the total number of times answered.</p>
</li>
<li>
<p>Once the user has gone through all 50 states, ask them if they'd like to play again.</p>
</li>
</ul>
<h2><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Getting Started</h2>
<p>You're given an array of dictionaries that contain each state name and capital.</p>
<blockquote>
